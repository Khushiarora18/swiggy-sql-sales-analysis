# swiggy-sql-sales-analysis
SQL project analyzing Swiggy food delivery data using data cleaning, star schema modeling, and business KPI analysis.

## Project Overview
This project analyzes Swiggy food delivery data using SQL.

The goal is to clean raw data, design a star schema data model, and generate business insights such as revenue trends, top cities, and popular dishes.

## Key Steps
- Data cleaning and validation
- Duplicate detection using ROW_NUMBER()
- Star schema dimensional modelling
- Business KPI analysis

## Technologies Used
- MySQL
- SQL
- Data Modelling
- Data Analysis

## Business Insights
- Monthly order trends
- Top performing cities
- Most ordered dishes
- Revenue by states
- Customer spending buckets
